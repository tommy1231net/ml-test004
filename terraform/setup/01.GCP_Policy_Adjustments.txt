### Policy Adjustments (Manual Provisioning)

After creating a new project,
the following organization policies were deactivated at the project level:

1) iam.allowedPolicyMemberDomains
   - Purpose: To allow unauthenticated (public) access to Cloud Run.

2) cloudbuild.useComputeServiceAccount
3) cloudbuild.useBuildServiceAccount
   - Purpose: To build the application using a custom service account.
   - Compliance: This change is in compliance with the following announcement regarding Cloud Build service account updates:
     https://cloud.google.com/build/docs/cloud-build-service-account-updates
